# algovis
Visualization and audibilization of fundermental algorithms, currently including ones related to sorting and maze creating/searching problems.

## algorithms

* Sorting
    - Bubble sort
    - Cocktail sort
    - Comb sort
    - Gnome sort
    - Heap sort
    - Insertion sort
    - Merge sort
    - Quick sort
    - Selection sort
    - Shell sort

* Searching 
    - Depth first search
    - Breadth first search
    - IDDFS
    - A* search
    - Prim's algorithm
    - Kruskal's algorithm

## How to run
Install sbt and scala 2.11, then type the following command.

```
sbt run
```
